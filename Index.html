How it works

The LSTM intelligence pipeline.

Every agent processes three real-time data streams, passes through a confidence gate, executes under strict risk controls, and retrains on every live outcome. A continuous loop that never stops.

01 — Input
Data Layer

Three simultaneous market signals

Real-time price action and OHLCV data. Order flow and order book depth. On-chain and news sentiment signals. All three feed the LSTM simultaneously, giving the model a complete picture of market state before a decision is made.

02 — Process
Model Layer

LSTM sequence model + LLM support

Long Short-Term Memory networks trained on 2+ years of real market data. Pattern recognition across all timeframes simultaneously. An LLM support layer for context interpretation. Confidence threshold gate: a trade only executes when the model is confident enough to act.

03 — Deploy
Agent Layer

100 independent agents, same intelligence

100 agents deploy simultaneously, each operating independently. Non-correlated execution. If one agent is in a position, others are not forced into the same trade. Diversification at the infrastructure level, not the portfolio level.

04 — Protect
Risk Layer

Hardcoded limits. No exceptions.

0.5% max position risk per trade. 3x maximum leverage. 5% daily drawdown cap triggers automatic pause. Stop-loss execution is automatic and cannot be overridden. Liquidation prevention is architecture, not policy. These rules are not guidelines — they are physics.

05 — Learn
Feedback Layer

Every trade improves the next signal

Trade outcomes are logged and fed back into the LSTM. The model retrains continuously on live results. Over time, agents become more accurate, more calibrated, more profitable. The intelligence compounds alongside the capital.

Risk Management

Hardcoded discipline. No exceptions.

Risk management is not a policy at IMMT. It is architecture. These parameters are enforced at the infrastructure level. No human can override them. No emotion can break them.

0.5%

Max risk per trade

Every trade is sized so that a full loss equals 0.5% of account value. Across 10 simultaneous positions, maximum exposure is 5%. This is hardcoded. Non-negotiable.

3x

Maximum leverage

Agents will not use more than 3x leverage regardless of signal confidence. This single constraint eliminates the majority of retail liquidation events.

5%

Daily drawdown cap

If daily drawdown reaches 5%, agents pause aggressive execution and shift to capital preservation mode. Trading resumes when drawdown reduces to 2%.

Auto

Stop-loss execution

Every position carries a predefined stop-loss that executes automatically. No "holding and hoping." No emotional override. If price hits the level, the agent exits.
